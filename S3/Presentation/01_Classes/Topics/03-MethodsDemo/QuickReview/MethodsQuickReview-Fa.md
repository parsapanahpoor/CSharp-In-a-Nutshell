# Methods - بررسی سریع

## تعریف
Method تابعی است که در داخل کلاس یا Struct تعریف می‌شود و رفتار (Behavior) یک نوع را پیاده‌سازی می‌کند.

## کاربرد
- پیاده‌سازی منطق و رفتار اشیاء
- کپسوله‌سازی عملیات قابل استفاده مجدد
- دستکاری و پردازش داده‌های کلاس
- ارائه Interface عمومی برای تعامل با شیء

## نکات کلیدی
- **Instance Methods**: به instance مربوطه دسترسی دارند (this)
- **Static Methods**: مستقل از instance هستند و به static members دسترسی دارند
- **Parameter Modifiers**: ref, out, in, params
- **Return Types**: می‌توانند مقدار برگردانند یا void باشند
- **Method Overloading**: امکان تعریف چند method با نام یکسان اما پارامترهای متفاوت
- **Optional Parameters**: پارامترهایی با مقدار پیش‌فرض
- **Named Arguments**: فراخوانی با نام پارامترها
- **Expression-bodied Methods**: سینتکس کوتاه‌تر با =>
- **Local Functions**: تعریف تابع داخل تابع (C# 7+)
- **Access Modifiers**: public, private, protected, internal, ...
- **Virtual/Override**: برای Polymorphism و Override در کلاس‌های مشتق
- **Extension Methods**: افزودن متد به انواع موجود بدون تغییر آن‌ها

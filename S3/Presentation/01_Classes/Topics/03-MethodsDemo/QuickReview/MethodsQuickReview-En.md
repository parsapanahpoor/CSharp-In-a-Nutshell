# Methods - Quick Review

## Definition
A method is a function defined inside a class or struct that implements the behavior of a type.

## Usage
- Implementing object logic and behavior
- Encapsulating reusable operations
- Manipulating and processing class data
- Providing public interface for object interaction

## Key Points
- **Instance Methods**: Have access to the instance (this)
- **Static Methods**: Independent of instance and access only static members
- **Parameter Modifiers**: ref, out, in, params
- **Return Types**: Can return a value or be void
- **Method Overloading**: Multiple methods with same name but different parameters
- **Optional Parameters**: Parameters with default values
- **Named Arguments**: Calling with parameter names
- **Expression-bodied Methods**: Shorter syntax with =>
- **Local Functions**: Defining function inside function (C# 7+)
- **Access Modifiers**: public, private, protected, internal, ...
- **Virtual/Override**: For Polymorphism and overriding in derived classes
- **Extension Methods**: Adding methods to existing types without modifying them

# وراثت 

## تعریف
کلاس اعضای کلاس پایه را به ارث می‌برد و رابطه ایجاد می‌کند. C# فقط **وراثت تکی** دارد.

## کاربرد
- استفاده مجدد کد
- چندریختی از طریق رفرنس پایه
- مدل‌سازی سلسله‌مراتب

## نکات کلیدی
- **وراثت تکی**: فقط یک کلاس پایه مستقیم
- **System.Object**: همه کلاس‌ها ضمناً از آن ارث می‌برند
- **به ارث نمی‌رسد**: سازنده‌ها، سازنده استاتیک، finalizer
- **sealed**: جلوی وراثت بعدی را می‌گیرد
- **Upcasting**: مشتق → پایه (ضمنی، امن)

<div dir="ltr" style="text-align: left;">

```csharp
class Base { }
class Derived : Base { }
Base b = new Derived();  // Upcast ضمنی ✅

```

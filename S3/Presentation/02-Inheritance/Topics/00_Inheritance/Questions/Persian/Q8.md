## ❓ Q8:

## می‌شه از Inheritance جلوگیری کرد؟ (sealed)

### ✅ جواب:

<div dir="ltr" style="text-align: left;">


```csharp
sealed class FinalClass {
// نمی‌شه ازش Inherit کرد
}

class MyClass : FinalClass { } // ❌ Error!

```

<div dir="rtl" style="text-align: right;">


### 🎯 کاربرد:

- **Security**: جلوگیری از Override رفتار حساس
- **Performance**: Compiler می‌تونه بهینه‌تر کنه (Devirtualization)
- **Design**: اطمینان از رفتار ثابت

**مثال واقعی:**

<div dir="ltr" style="text-align: left;">


```csharp
public sealed class String { } // String در .NET sealed است!
```

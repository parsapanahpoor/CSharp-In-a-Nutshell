## ❓ Q3:

## Access Modifiers (public, protected, private) چطور کار می‌کنند؟

### ✅ جواب:

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
public int PublicField;        // همه می‌بینند ✅
protected int ProtectedField;  // فقط Derived ✅
private int PrivateField;      // فقط Base ❌
}

class Derived : Base {
public void Test() {
PublicField = 1;      // ✅ OK
ProtectedField = 2;   // ✅ OK
PrivateField = 3;     // ❌ Error!
}
}

// خارج از کلاس:
var obj = new Derived();
obj.PublicField = 1;      // ✅ OK
obj.ProtectedField = 2;   // ❌ Error
obj.PrivateField = 3;     // ❌ Error

```

<div dir="rtl" style="text-align: right;">

### 🎯 کاربرد:

- **public**: API عمومی
- **protected**: Extension Points (برای Override)
- **private**: Implementation Details

## ❓ Q9:

## تفاوت Inheritance با Composition چیه؟ کی کدوم رو استفاده کنیم؟

### ✅ Inheritance (IS-A):

<div dir="ltr" style="text-align: left;">


```csharp
// Dog IS-A Animal ✅
class Animal { }
class Dog : Animal { }

```

<div dir="rtl" style="text-align: right;">


### ✅ Composition (HAS-A):

<div dir="ltr" style="text-align: left;">


```csharp
// Car HAS-A Engine ✅
class Engine { }
class Car {
private Engine _engine; // Composition
}

```

<div dir="rtl" style="text-align: right;">


### ❌ اشتباه رایج:

<div dir="ltr" style="text-align: left;">


```csharp
// ❌ Car IS-A Engine?! نه!
class Car : Engine { } // Wrong!

```

<div dir="rtl" style="text-align: right;">


### 🎯 قانون:

- **IS-A** → Inheritance
- **HAS-A** → Composition
- **شک داری؟** → Composition را انتخاب کن! (Favor Composition Over Inheritance)

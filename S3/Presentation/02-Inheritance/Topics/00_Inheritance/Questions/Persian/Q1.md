## ❓ Q1:

## چرا اصلاً به Inheritance نیاز داریم؟

### 💡 مشکل:

<div dir="ltr" style="text-align: left;">

```csharp
// 3 نوع کارمند داریم، کد تکراری زیاد:
class FullTimeEmployee {
public string Name;
public void ClockIn() { /* ... */ }
}

class PartTimeEmployee {
public string Name;      // تکراری!
public void ClockIn() { /* ... */ } // تکراری!
}
```

<div dir="rtl" style="text-align: right;">
### ✅ جواب:

<div dir="ltr" style="text-align: left;">


```csharp
class Employee {
public string Name;
public void ClockIn() { /* ... */ }
}

class FullTimeEmployee : Employee { }
class PartTimeEmployee : Employee { }

// ✅ Code Reuse
// ✅ Maintainability
// ✅ Polymorphism
```
<div dir="rtl" style="text-align: right;">

### 🎯 نکته: Inheritance = DRY Principle (Don't Repeat Yourself)

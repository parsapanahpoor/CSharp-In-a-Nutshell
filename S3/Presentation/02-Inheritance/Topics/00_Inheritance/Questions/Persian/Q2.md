## ❓ Q2:

## Syntax دقیق چیه؟ چطوری Inherit کنیم؟

### ✅ جواب:

<div dir="ltr" style="text-align: left;">


```csharp
class BaseClass {
public int Value { get; set; }
}

class DerivedClass : BaseClass {
//                  ↑
//         اینجا از BaseClass ارث می‌بریم
}

// استفاده:
var obj = new DerivedClass();
obj.Value = 10; // ✅ Value از BaseClass اومده

```

<div dir="rtl" style="text-align: right;">

### 🎯 نکته کلیدی:

- فقط یک کلاس می‌تونیم Inherit کنیم (Single Inheritance)
- اما چند Interface مجاز است:

<div dir="ltr" style="text-align: left;">


```csharp
class MyClass : BaseClass, IInterface1, IInterface2 { }
//              ↑           ↑
//           کلاس       Interfaces
```

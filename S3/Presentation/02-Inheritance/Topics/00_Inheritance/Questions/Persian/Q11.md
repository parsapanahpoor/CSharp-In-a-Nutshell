## ❓ Q11:

## چرا base.base نداریم؟ چطوری به GrandParent دسترسی پیدا کنیم؟

### 💡 مشکل:

<div dir="ltr" style="text-align: left;">

```csharp
class GrandParent {
public virtual void Method() => Console.WriteLine("GrandParent");
}

class Parent : GrandParent {
public override void Method() => Console.WriteLine("Parent");
}

class Child : Parent {
public override void Method() {
base.Method();           // ✅ Parent رو صدا می‌زنه
base.base.Method();      // ❌ Syntax Error!

// چطوری GrandParent رو صدا کنم؟ 🤔
}
}

<div dir="rtl" style="text-align: right;">
```

### ✅ جواب:

<div dir="ltr" style="text-align: left;">

```csharp
// راه حل 1: از Parent استفاده کن
class Parent : GrandParent {
public override void Method() => Console.WriteLine("Parent");

protected void CallGrandParentMethod() {
base.Method(); // GrandParent
}
}

class Child : Parent {
public override void Method() {
CallGrandParentMethod(); // ✅ Indirect Access
}
}

// راه حل 2: Template Method Pattern
class GrandParent {
public void Process() {
Step1();
Step2();
}
protected virtual void Step1() { }
protected virtual void Step2() { }
}

class Child : Parent {
protected override void Step1() {
// فقط این قسمت رو Override می‌کنیم
}
}
```

<div dir="rtl" style="text-align: right;">

### 🎯 دلیل طراحی:

- `base.base` نداریم چون **Encapsulation** رو نقض می‌کنه
- هر کلاس فقط باید با Parent مستقیمش کانه
- اگه نیاز به GrandParent داری → طراحی اشتباهه!

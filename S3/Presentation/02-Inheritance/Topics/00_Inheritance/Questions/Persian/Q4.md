## ❓ Q4:

## چرا C# Multiple Inheritance نداره؟ مشکلش چیه؟

### 💡 مشکل: Diamond Problem

<div dir="ltr" style="text-align: left;">


```csharp
class A {
public void Method() => Console.WriteLine("A");
}

class B : A {
public new void Method() => Console.WriteLine("B");
}

class C : A {
public new void Method() => Console.WriteLine("C");
}

// ❌ اگه Multiple Inheritance داشتیم:
class D : B, C {  // کدوم Method() رو صدا کنیم؟!
public void Test() {
Method(); // B.Method یا C.Method؟ 🤔
}
}

```

<div dir="rtl" style="text-align: right;">


### ✅ راه‌حل C#:

<div dir="ltr" style="text-align: left;">


```csharp
interface IB { void MethodB(); }
interface IC { void MethodC(); }

class D : A, IB, IC {  // ✅ یه کلاس + چند Interface
public void MethodB() { /* ... */ }
public void MethodC() { /* ... */ }
}

```

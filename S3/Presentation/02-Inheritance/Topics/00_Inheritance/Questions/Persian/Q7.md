## ❓ Q7:

## Constructor Chaining چیه؟ ترتیب اجرا چطوریه؟

### ✅ جواب:

<div dir="ltr" style="text-align: left;">


```csharp
class Base {
public Base() {
Console.WriteLine("1. Base Constructor");
}
}

class Derived : Base {
public Derived() {
Console.WriteLine("2. Derived Constructor");
}
}

var obj = new Derived();

// خروجی:
// 1. Base Constructor
// 2. Derived Constructor

```

<div dir="rtl" style="text-align: right;">


### 🎯 قانون:

- همیشه Base اول اجرا می‌شود
- بعد Derived
- دلیل: فیلدهای Base باید اول Initialize بشوند

**پیچیده‌تر:**

<div dir="ltr" style="text-align: left;">


```csharp
class Base {
public Base(int x) {
Console.WriteLine($"Base({x})");
}
}

class Derived : Base {
public Derived(int x, int y) : base(x) { // ✅ باید صریح Base رو صدا کنیم
Console.WriteLine($"Derived({y})");
}
}
```

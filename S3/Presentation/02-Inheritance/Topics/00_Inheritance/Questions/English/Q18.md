### ğŸ¯ Advanced Points:

1. **GetType()** is not virtual - always returns the actual type
2. **MemberwiseClone()** is protected - only accessible from within the class
3. **Finalize()** is protected virtual - used for destructor
4. **ReferenceEquals()** is a static method - reference comparison

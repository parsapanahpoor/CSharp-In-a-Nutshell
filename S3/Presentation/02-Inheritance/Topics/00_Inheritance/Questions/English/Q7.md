## What is Constructor Chaining? What is the execution order?

### âœ… Answer:

<div dir="ltr" style="text-align: left;">


```csharp
class Base {
public Base() {
Console.WriteLine("1. Base Constructor");
}
}

class Derived : Base {
public Derived() {
Console.WriteLine("2. Derived Constructor");
}
}

var obj = new Derived();

// Output:
// 1. Base Constructor
// 2. Derived Constructor

```

<div dir="ltr" style="text-align: left;">


### ğŸ¯ Rule:

- Base always executes first
- Then Derived
- Reason: Base fields must be initialized first

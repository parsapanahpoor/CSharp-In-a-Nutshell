## ❓ Q15:

## What is Covariance/Contravariance in Inheritance?

## ✅ Answer:

<div dir="ltr" style="text-align: left;">


```csharp
// Covariance: Return Type can be Derived
class Animal { }
class Dog : Animal { }

class AnimalShelter {
public virtual Animal GetAnimal() => new Animal();
}

class DogShelter : AnimalShelter {
public override Dog GetAnimal() => new Dog(); // ✅ C# 9+ (Covariant Return Types)
}

```

<div dir="ltr" style="text-align: left;">


## ❌ Before C# 9:

<div dir="ltr" style="text-align: left;">


```csharp
class AnimalShelter {
public virtual Animal GetAnimal() => new Animal();
}

class DogShelter : AnimalShelter {
public override Animal GetAnimal() => new Dog(); // ❌ Must return Animal
}

```

<div dir="ltr" style="text-align: left;">


## Contravariance (Parameter Types):

<div dir="ltr" style="text-align: left;">


```csharp
// ❌ C# doesn't allow:
class Base {
public virtual void Process(Dog dog) { }
}

class Derived : Base {
public override void Process(Animal animal) { } // ❌ Error: Contravariance not supported
}

// Why? Type Safety problem:
Base obj = new Derived();
obj.Process(new Dog()); // Derived.Process() expects Animal, but receives Dog!
```

## What is Sealed Override?

**Sealed Override** means **sealing** an overridden method so it cannot be overridden anymore.

```csharp
class Base {
    public virtual void Method() { }
}

class Derived : Base {
    public sealed override void Method() {  // âœ… Sealed override
        // ...
    }
}

class Derived2 : Derived {
    // public override void Method() { }  // âŒ Error! Cannot override sealed method
}
```

## How do Sealed Classes work with Reflection and Runtime Type Information?

### ‚úÖ Deep Answer:

Reflection can inspect Sealed Classes at runtime and read Type Information.

### üîç Reflection for Sealed Classes:

<div dir="ltr" style="text-align: left;">

```csharp
sealed class FinalClass {
    public void Method() { }
}

// Reflection:
Type type = typeof(FinalClass);
bool isSealed = type.IsSealed;  // True ‚úÖ
bool canInherit = !type.IsSealed;  // False

// Check Sealed Methods:
MethodInfo method = type.GetMethod("Method");
bool isSealedMethod = method.IsFinal;  // If sealed override
```

### üî¨ Type Information:

<div dir="ltr" style="text-align: left;">

```csharp
sealed class Dog : Animal {
    public sealed override void Speak() { }
}

Type type = typeof(Dog);

// Check Sealed Class:
Console.WriteLine(type.IsSealed);  // True ‚úÖ

// Check Sealed Methods:
MethodInfo[] methods = type.GetMethods();
foreach (var method in methods) {
    if (method.IsFinal) {
        Console.WriteLine($"Sealed: {method.Name}");  // Speak ‚úÖ
    }
}
```

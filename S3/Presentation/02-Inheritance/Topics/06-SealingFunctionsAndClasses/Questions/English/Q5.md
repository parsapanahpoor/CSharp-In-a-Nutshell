## What is Sealed Methods Performance Optimization?

### âœ… Answer:

Sealed Methods allow compiler to make **Direct Call** instead of Virtual Call.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public virtual void Method() { }
}

class Derived : Base {
    public sealed override void Method() { }  // âœ… Sealed
}

Base obj = new Derived();
obj.Method();  // Compiler can make Direct Call! (Optimization)
```

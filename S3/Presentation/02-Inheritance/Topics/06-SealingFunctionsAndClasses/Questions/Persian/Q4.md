## ❓ Q4:

## Sealed Class می‌تونه Abstract باشه؟

### ❌ جواب: خیر!

Abstract و Sealed **متناقض** هستند. Abstract می‌گه "باید ارث‌بری بشه" و Sealed می‌گه "نمی‌تونه ارث‌بری بشه".

<div dir="ltr" style="text-align: left;">

```csharp
// ❌ نمی‌تونیم این کار رو بکنیم:
// abstract sealed class Animal { }  // Error!
```

<div dir="rtl" style="text-align: right;">

### 🎯 نکته:

Abstract و Sealed **ناسازگار** هستند. نمی‌تونیم همزمان استفاده کنیم.

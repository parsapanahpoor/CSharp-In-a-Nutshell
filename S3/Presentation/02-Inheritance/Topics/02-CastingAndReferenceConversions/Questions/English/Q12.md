## Why doesn't `as` work with value types?

### âœ… Answer:

Because `as` returns `null` on failure, but value types cannot be `null`!

<div dir="ltr" style="text-align: left;">

```csharp
int number = 42;
// int? result = number as int?;  // âŒ Error: as cannot work with value types

// Solution: Use is
if (number is int i) {
    Console.WriteLine(i);  // âœ…
}

// Or with nullable cast:
int? nullable = number as int?;  // âŒ Still doesn't work!
```

<div dir="ltr" style="text-align: left;">

### âœ… Solution for Value Types:

<div dir="ltr" style="text-align: left;">

```csharp
object obj = 42;

// Method 1: With is
if (obj is int i) {
    Console.WriteLine(i);  // âœ…
}

// Method 2: Direct cast (with try-catch)

    int num = (int)obj;  // âœ…
} catch (InvalidCastException) {
    // handle
}
```

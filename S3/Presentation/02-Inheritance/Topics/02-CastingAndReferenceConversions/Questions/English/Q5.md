## What is the `is` operator?

### âœ… Answer:

The `is` operator is used for **type checking**. It can be combined with **pattern matching**.

<div dir="ltr" style="text-align: left;">

```csharp
class Animal { }
class Dog : Animal { }
class Cat : Animal { }

Animal animal = new Dog();

// Old way:
if (animal is Dog) {
    Dog dog = (Dog)animal;
    dog.Bark();
}

// New way (Pattern Matching - C# 7+):
if (animal is Dog d) {
    d.Bark();  // âœ… d is already Dog!
}
```

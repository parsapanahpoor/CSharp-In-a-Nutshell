## How does Pattern Matching with `is` work?

### âœ… Answer:

Pattern Matching in C# 7+ allows us to do **check and cast** in one line.

<div dir="ltr" style="text-align: left;">

```csharp
class Animal { }
class Dog : Animal {
    public string Name { get; set; }
}
class Cat : Animal {
    public int Age { get; set; }
}

Animal animal = new Dog { Name = "Buddy" };

// Pattern Matching:
if (animal is Dog d) {
    Console.WriteLine(d.Name);  // âœ… d is already Dog
}

// With switch expression (C# 8+):
string sound = animal switch {
    Dog dog => $"Dog {dog.Name} barks",
    Cat cat => $"Cat is {cat.Age} years old",
    _ => "Unknown animal"
};
```

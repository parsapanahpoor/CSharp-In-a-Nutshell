## What is InvalidCastException?

### ‚úÖ Answer:

When we try to cast an object to a type it **cannot** be converted to, this exception is thrown.

<div dir="ltr" style="text-align: left;">

```csharp
class Animal { }
class Dog : Animal { }
class Cat : Animal { }

Animal animal = new Cat();

try {
    Dog dog = (Dog)animal;  // ‚ùå InvalidCastException!
    // Cat cannot be converted to Dog
} catch (InvalidCastException ex) {
    Console.WriteLine($"Cast failed: {ex.Message}");
}
```

<div dir="ltr" style="text-align: left;">

### üéØ Safe Solutions:

<div dir="ltr" style="text-align: left;">

```csharp
// Method 1: With as
Dog? dog = animal as Dog;
if (dog != null) {
    dog.Bark();
}

// Method 2: With is
if (animal is Dog d) {
    d.Bark();
}
```

## What is Upcasting?

### âœ… Answer:

**Upcasting** means converting from derived class to base class. This conversion is **implicit** and **always safe**.

<div dir="ltr" style="text-align: left;">

```csharp
class Animal { }
class Dog : Animal { }

Dog dog = new Dog();
Animal animal = dog;  // âœ… Upcast (Implicit - doesn't need cast)

// Why is it safe? Because every Dog is an Animal!
```

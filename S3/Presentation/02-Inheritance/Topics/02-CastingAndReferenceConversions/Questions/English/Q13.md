## How can we check multiple types?

### âœ… Answer:

With `is` pattern matching we can check multiple types.

<div dir="ltr" style="text-align: left;">

```csharp
class Animal { }
class Dog : Animal { }
class Cat : Animal { }
class Bird : Animal { }

Animal animal = new Dog();

// Method 1: if-else chain
if (animal is Dog d) {
    d.Bark();
} else if (animal is Cat c) {
    c.Meow();
} else if (animal is Bird b) {
    b.Fly();
}

// Method 2: switch expression (C# 8+)
string action = animal switch {
    Dog dog => "Barking",
    Cat cat => "Meowing",
    Bird bird => "Flying",
    _ => "Unknown"
};
```

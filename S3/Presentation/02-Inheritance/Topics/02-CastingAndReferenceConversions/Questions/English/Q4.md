## What is the `as` operator?

### ‚úÖ Answer:

The `as` operator performs a **safe cast**. If it fails, it returns `null` (without exception).

<div dir="ltr" style="text-align: left;">

```csharp
class Animal { }
class Dog : Animal { }
class Cat : Animal { }

Animal animal = new Cat();

Dog? dog = animal as Dog;  // ‚úÖ Returns null (not exception)
if (dog != null) {
    dog.Bark();
} else {
    Console.WriteLine("Not a Dog!");
}
```

<div dir="ltr" style="text-align: left;">

### üéØ Note:

`as` only works with **reference types**. It doesn't work with value types.

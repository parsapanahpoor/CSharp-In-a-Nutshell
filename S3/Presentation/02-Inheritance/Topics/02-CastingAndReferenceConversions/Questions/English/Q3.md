## What is Downcasting?

### ‚úÖ Answer:

**Downcasting** means converting from base class to derived class. This conversion is **explicit** and may **fail**.

<div dir="ltr" style="text-align: left;">

```csharp
class Animal { }
class Dog : Animal { }
class Cat : Animal { }

Animal animal = new Dog();
Dog dog = (Dog)animal;  // ‚úÖ Downcast (Explicit - needs cast)

Animal animal2 = new Cat();
Dog dog2 = (Dog)animal2;  // ‚ùå InvalidCastException!
```

<div dir="ltr" style="text-align: left;">

### üéØ Note:

Downcasting is **dangerous**! We must make sure the object is actually of the desired type.

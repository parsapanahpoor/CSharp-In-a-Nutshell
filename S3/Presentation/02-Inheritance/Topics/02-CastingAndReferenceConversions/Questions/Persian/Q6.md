## ❓ Q6:

## تفاوت `as` و `is` چیه؟

### ✅ مقایسه:

| ویژگی | `as` | `is` |
|-------|------|------|
| **نوع خروجی** | نوع مورد نظر یا `null` | `bool` |
| **Exception** | نمی‌ده | نمی‌ده |
| **Pattern Matching** | ندارد | دارد (C# 7+) |
| **استفاده** | وقتی می‌خوایم کست کنیم | وقتی می‌خوایم بررسی کنیم |

<div dir="ltr" style="text-align: left;">

```csharp
Animal animal = new Dog();

// با as:
Dog? dog = animal as Dog;
if (dog != null) {
    dog.Bark();
}

// با is (Pattern Matching):
if (animal is Dog d) {
    d.Bark();  // ✅ کوتاه‌تر و خوانا‌تر
}
```

<div dir="rtl" style="text-align: right;">

### 🎯 نکته:

در C# 7+، `is` با pattern matching **بهتر** از `as` است چون کوتاه‌تر و ایمن‌تره.

## ❓ Q1:

## Casting یعنی چی؟

### ✅ جواب:

**Casting** یعنی تبدیل رفرنس از یک نوع به نوع دیگر در سلسله‌مراتب Inheritance.

<div dir="ltr" style="text-align: left;">

```csharp
class Animal { }
class Dog : Animal { }

Dog dog = new Dog();
Animal animal = dog;  // ✅ Casting (Upcast)
```

<div dir="rtl" style="text-align: right;">

### 🎯 نکته:

Casting فقط **نوع رفرنس** رو تغییر می‌ده، نه خود شی رو! شی همون شی می‌مونه.

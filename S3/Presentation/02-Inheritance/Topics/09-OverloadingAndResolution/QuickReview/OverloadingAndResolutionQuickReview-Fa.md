## تعریف
بارگذاری بیش از حد متد اجازه می‌دهد چند متد با نام یکسان اما پارامترهای متفاوت داشته باشیم. کامپایلر بهترین تطبیق را در زمان کامپایل انتخاب می‌کند.

## کاربرد
متدها را بر اساس نوع، تعداد یا ترتیب پارامتر بارگذاری کنید:

<div dir="ltr" style="text-align: left;">

```csharp
public class Math
{
public int Add(int a, int b) { }
public double Add(double a, double b) { }
public int Add(int a, int b, int c) { }
}
```

<div dir="rtl" style="text-align: right;">

## نکات کلیدی
- متدها باید در پارامترها (نوع، تعداد یا ترتیب) متفاوت باشند
- نوع بازگشتی به تنهایی نمی‌تواند بارگذاری‌ها را تشخیص دهد
- کامپایلر خاص‌ترین تطبیق را انتخاب می‌کند
- تفکیک در زمان کامپایل اتفاق می‌افتد
- متدهای ارث‌بری شده در تفکیک بارگذاری شرکت می‌کنند
- پارامترهای `params` کمترین اولویت را دارند
- Null با انواع مرجع تطبیق می‌یابد، نه انواع مقداری

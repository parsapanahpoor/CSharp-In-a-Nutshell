## When to use Overloading?

### âœ… Answer:

When we want to perform **same operation** with **different types**.

### Examples:

1. **Math Operations**: `Add(int)`, `Add(double)`, `Add(decimal)`
2. **String Operations**: `Format(string)`, `Format(string, object)`
3. **Collection Operations**: `Add(item)`, `AddRange(items)`

<div dir="ltr" style="text-align: left;">

```csharp
class Calculator {
    public int Add(int a, int b) => a + b;
    public double Add(double a, double b) => a + b;
    public decimal Add(decimal a, decimal b) => a + b;
}
```

## How does Overloading work with Generic Methods?

### âœ… Answer:

Generic Methods can also be overloaded, but Type Inference is important.

<div dir="ltr" style="text-align: left;">

```csharp
class Test {
    public void Method<T>(T item) {
        Console.WriteLine($"Generic: {typeof(T).Name}");
    }
    
    public void Method(int x) {
        Console.WriteLine("Int");
    }
}

var test = new Test();
test.Method(5);  // Method(int) is selected âœ… (Non-generic has priority)
test.Method("Hi");  // Method<string>(string) âœ…
```

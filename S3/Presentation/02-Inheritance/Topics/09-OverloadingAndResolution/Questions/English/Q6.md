## How does Best Match Algorithm work?

### âœ… Answer:

Compiler calculates **Cost** of each match and selects **lowest cost**.

<div dir="ltr" style="text-align: left;">

```csharp
class Test {
    public void Method(int x) { }      // Cost = 0 (Exact match)
    public void Method(object x) { }    // Cost = 1 (Implicit conversion)
}

var test = new Test();
test.Method(5);  // Method(int) is selected âœ… (Cost = 0)
```

## How does Overloading work with Inheritance?

### âœ… Answer:

**Inherited** methods also participate in Overload Resolution.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public void Process(int x) {
        Console.WriteLine("Base: int");
    }
}

class Derived : Base {
    public void Process(string s) {
        Console.WriteLine("Derived: string");
    }
}

Derived d = new Derived();
d.Process(10);   // Base.Process(int) âœ… (inherited)
d.Process("Hi"); // Derived.Process(string) âœ…
```

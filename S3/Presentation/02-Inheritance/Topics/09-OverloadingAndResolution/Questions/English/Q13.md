## ‚ùì Q13:

## How does Overloading work with Operator Overloading?

### ‚úÖ Answer:

Operator Overloading works like Method Overloading.

<div dir="ltr" style="text-align: left;">

```csharp
class Vector {
    public int X, Y;
    
    public static Vector operator +(Vector a, Vector b) {
        return new Vector { X = a.X + b.X, Y = a.Y + b.Y };
    }
    
    public static Vector operator +(Vector a, int scalar) {
        return new Vector { X = a.X + scalar, Y = a.Y + scalar };
    }
}

var v1 = new Vector { X = 1, Y = 2 };
var v2 = new Vector { X = 3, Y = 4 };
var v3 = v1 + v2;  // operator+(Vector, Vector) ‚úÖ
var v4 = v1 + 5;   // operator+(Vector, int) ‚úÖ
```

## ❓ Q2:

## Method Resolution چیه؟

### ✅ جواب:

**Method Resolution** یعنی کامپایلر **بهترین متد** رو از بین Overload ها انتخاب می‌کنه.

<div dir="ltr" style="text-align: left;">

```csharp
class Math {
    public int Add(int a, int b) => a + b;
    public double Add(double a, double b) => a + b;
}

var math = new Math();
math.Add(5, 3);  // کامپایلر Add(int, int) رو انتخاب می‌کنه ✅
```

<div dir="rtl" style="text-align: right;">

### 🎯 نکته:

Resolution در **Compile-time** انجام می‌شه. کامپایلر بهترین match رو پیدا می‌کنه.

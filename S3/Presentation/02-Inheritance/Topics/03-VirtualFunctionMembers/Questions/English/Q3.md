## What is Virtual Property?

### âœ… Answer:

**Virtual Property** works like Virtual Method. We can make a property virtual in base class and override it in derived.

<div dir="ltr" style="text-align: left;">

```csharp
class Employee {
    public double BaseSalary { get; set; }
    
    public virtual double TotalSalary {
        get => BaseSalary;
    }
}

class Manager : Employee {
    public override double TotalSalary {
        get => BaseSalary * 1.5;  // 50% bonus
    }
}

Employee emp = new Manager { BaseSalary = 5000 };
Console.WriteLine(emp.TotalSalary);  // 7500 âœ…
```

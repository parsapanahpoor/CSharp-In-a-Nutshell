## What is the difference between Virtual Methods and Abstract Methods?

### âœ… Comparison:

| Feature | Virtual | Abstract |
|-------|---------|----------|
| **Implementation** | Must have | Must not have |
| **Class** | Can be concrete | Must be abstract |
| **Override** | Optional | Mandatory |
| **Usage** | Default behavior | Contract |

    public virtual void Speak() {
        Console.WriteLine("Animal sound");  // âœ… Default implementation

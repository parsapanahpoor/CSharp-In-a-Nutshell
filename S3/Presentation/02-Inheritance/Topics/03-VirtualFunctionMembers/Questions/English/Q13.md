## How do Virtual Methods work with Events?

### âœ… Answer:

Events cannot be virtual, but we can use **virtual methods** to raise events.

<div dir="ltr" style="text-align: left;">

```csharp
class Animal {
    public event EventHandler? Spoke;
    
    protected virtual void OnSpoke() {  // âœ… Virtual method
        Spoke?.Invoke(this, EventArgs.Empty);
    }
    
    public void Speak() {
        Console.WriteLine("Animal sound");
        OnSpoke();  // âœ… Virtual method is called
    }
}

class Dog : Animal {
    protected override void OnSpoke() {  // âœ… Override
        Console.WriteLine("Dog is about to speak...");
        base.OnSpoke();  // âœ… Raises base event
    }
}

var dog = new Dog();
dog.Spoke += (s, e) => Console.WriteLine("Event fired!");
dog.Speak();
// Output:
// Animal sound
// Dog is about to speak...
// Event fired!
```

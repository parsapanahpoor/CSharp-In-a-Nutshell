## ‚ùì Q4:

## What is base keyword?

### ‚úÖ Answer:

`base` keyword allows us to access **base class method or property**.

<div dir="ltr" style="text-align: left;">

```csharp
class Printer {
    public virtual void Print(string text) {
        Console.WriteLine($"[Base] Printing: {text}");
    }
}

class AdvancedPrinter : Printer {
    public override void Print(string text) {
        Console.WriteLine("[Advanced] Pre-processing...");
        base.Print(text);  // ‚úÖ Calling base method
        Console.WriteLine("[Advanced] Post-processing...");
    }
}

var printer = new AdvancedPrinter();
printer.Print("Hello");
// Output:
// [Advanced] Pre-processing...
// [Base] Printing: Hello
// [Advanced] Post-processing...
```

<div dir="ltr" style="text-align: left;">

### üéØ Note:

`base` only works for **override methods**. We cannot use base for regular methods.

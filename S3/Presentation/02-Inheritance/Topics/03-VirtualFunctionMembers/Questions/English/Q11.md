## Can we make a virtual method virtual again?

When we override a virtual method, it's **no longer virtual** unless we explicitly write `virtual` again.

```csharp

    public virtual void Method() { }
}

class Derived : Base {
    public override void Method() { }  // âœ… Overridden
    // But not virtual! Cannot be overridden again
}

class Derived2 : Derived {
    // public override void Method() { }  // âŒ Error! Method is not virtual
}
```

<div dir="ltr" style="text-align: left;">

```csharp
    public virtual override void Method() { }  // âœ… Virtual again
}

    public override void Method() { }  // âœ… Now we can override
}
```

## How do Virtual Methods work with Indexers?

### âœ… Answer:

Indexers can also be virtual and overridden.

<div dir="ltr" style="text-align: left;">

```csharp
class Collection {
    protected int[] items = new int[10];
    
    public virtual int this[int index] {
        get => items[index];
        set => items[index] = value;
    }
}

class SafeCollection : Collection {
    public override int this[int index] {
        get {
            if (index < 0 || index >= items.Length) {
                throw new IndexOutOfRangeException();
            }
            return base[index];  // âœ… base indexer
        }
        set {
            if (index < 0 || index >= items.Length) {
                throw new IndexOutOfRangeException();
            }
            base[index] = value;  // âœ… base indexer
        }
    }
}

var collection = new SafeCollection();
collection[0] = 42;  // âœ…
Console.WriteLine(collection[0]);  // 42 âœ…
```

## تعریف
سازنده‌ها از کلاس پایه به کلاس مشتق در سلسله‌مراتب وراثت اجرا می‌شوند.

## کاربرد
از `: base()` برای فراخوانی سازنده پایه استفاده کنید:

<div dir="ltr" style="text-align: left;">

```csharp
public class Base
{
public Base(int x) { }
}

public class Derived : Base
{
public Derived(int x) : base(x) { }  // باید پایه را فراخوانی کند
}
```

<div dir="rtl" style="text-align: right;">

## نکات کلیدی
- سازنده پایه قبل از سازنده مشتق اجرا می‌شود
- اگر سازنده بدون پارامتر وجود نداشته باشد باید سازنده پایه فراخوانی شود
- از `: base(args)` برای ارسال پارامتر به سازنده پایه استفاده کنید
- ترتیب اجرا: پدربزرگ ← والد ← فرزند
- سازنده‌ها قابل ارث‌بری نیستند
- سازنده‌های static قبل از instance اجرا می‌شوند
- از `: this()` برای فراخوانی سازنده دیگر در همان کلاس استفاده کنید

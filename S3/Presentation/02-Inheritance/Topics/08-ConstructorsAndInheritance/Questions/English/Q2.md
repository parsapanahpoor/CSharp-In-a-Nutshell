## What is the execution order of Constructors?

### âœ… Answer:

Constructors execute from **Base to Derived**. Base Constructor is called first.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public Base() {
        Console.WriteLine("Base");
    }
}

class Derived : Base {
    public Derived() {
        Console.WriteLine("Derived");
    }
}

new Derived();
// Output:
// Base
// Derived
```

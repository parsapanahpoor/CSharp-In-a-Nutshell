## How do Constructors work in Inheritance?

### âœ… Answer:

Constructors are **not inherited**! Each class must have its own constructor.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public Base() {
        Console.WriteLine("Base constructor");
    }
}

class Derived : Base {
    public Derived() {  // âœ… Must have constructor
        Console.WriteLine("Derived constructor");
    }
}
```

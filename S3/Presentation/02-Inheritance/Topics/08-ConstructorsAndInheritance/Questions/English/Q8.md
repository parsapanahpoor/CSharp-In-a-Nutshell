## Can we make Constructor private?

### âœ… Answer:

Yes! Private Constructor is used for **Singleton Pattern** or **Static Classes**.

<div dir="ltr" style="text-align: left;">

```csharp
class Singleton {
    private Singleton() { }  // âœ… Private constructor
    
    private static Singleton instance;
    public static Singleton Instance {
        get {
            if (instance == null) {
                instance = new Singleton();
            }
            return instance;
        }
    }
}
```

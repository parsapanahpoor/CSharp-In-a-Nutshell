## How do Static Constructors work in Inheritance?

### âœ… Answer:

Static Constructors execute from **Base to Derived**, before Instance Constructors.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    static Base() {
        Console.WriteLine("Base static");
    }
}

class Derived : Base {
    static Derived() {
        Console.WriteLine("Derived static");
    }
}

new Derived();
// Output:
// Base static
// Derived static
```

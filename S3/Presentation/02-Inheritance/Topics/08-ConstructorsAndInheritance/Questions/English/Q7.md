## How do Constructors work in Multi-Level Inheritance?

### âœ… Answer:

Constructors execute from **GrandParent to Parent to Child**.

<div dir="ltr" style="text-align: left;">

```csharp
class GrandParent {
    public GrandParent() {
        Console.WriteLine("GrandParent");
    }
}

class Parent : GrandParent {
    public Parent() {
        Console.WriteLine("Parent");
    }
}

class Child : Parent {
    public Child() {
        Console.WriteLine("Child");
    }
}

new Child();
// Output:
// GrandParent
// Parent
// Child
```

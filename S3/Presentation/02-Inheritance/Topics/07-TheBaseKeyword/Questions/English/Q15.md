## What is the order of base in Constructor?

### âœ… Answer:

`base` in Constructor is called **before body**. Base Constructor executes first.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public Base() {
        Console.WriteLine("Base constructor");
    }
}

class Derived : Base {
    public Derived() : base() {  // âœ… base is called first
        Console.WriteLine("Derived constructor");
    }
}

new Derived();
// Output:
// Base constructor
// Derived constructor
```

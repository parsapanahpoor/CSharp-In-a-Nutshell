## How does base work in Multi-Level Inheritance?

### âœ… Answer:

`base` only refers to **direct Base Class**. Cannot reach GrandParent.

<div dir="ltr" style="text-align: left;">

```csharp
class GrandParent {
    public virtual void Method() {
        Console.WriteLine("GrandParent");
    }
}

class Parent : GrandParent {
    public override void Method() {
        Console.WriteLine("Parent");
    }
}

class Child : Parent {
    public override void Method() {
        base.Method();  // âœ… Parent.Method() (direct Base)
        // For GrandParent must use Cast:
        ((GrandParent)this).Method();  // âœ… GrandParent.Method()
    }
}
```

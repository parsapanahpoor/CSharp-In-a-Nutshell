## How does base work in Interface Implementation?

### âœ… Answer:

`base` in Interface Implementation **doesn't work** because Interface doesn't have implementation!

<div dir="ltr" style="text-align: left;">

```csharp
interface IAnimal {
    void Speak();
}

class Animal : IAnimal {
    public void Speak() {
        Console.WriteLine("Animal");
    }
}

class Dog : Animal, IAnimal {
    public void Speak() {
        // base.Speak();  // âœ… Animal.Speak() (from Base Class)
        // But base doesn't refer to Interface!
    }
}
```

## How does base work in Constructor Chaining?

### âœ… Answer:

`base` in Constructor Chaining is used to **call Base Constructor**.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public Base() {
        Console.WriteLine("Base()");
    }
    
    public Base(string name) {
        Console.WriteLine($"Base({name})");
    }
}

class Derived : Base {
    public Derived() : base() {  // âœ… Base()
        Console.WriteLine("Derived()");
    }
    
    public Derived(string name) : base(name) {  // âœ… Base(name)
        Console.WriteLine($"Derived({name})");
    }
}
```

## How does base work in Static Methods?

### âŒ Answer: It doesn't work!

`base` doesn't work in Static Methods because Static Methods don't belong to instance.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public static void Method() { }
}

class Derived : Base {
    public static void Method() {
        // base.Method();  // âŒ Error! base doesn't work in static
        Base.Method();  // âœ… Must call Base.Method() directly
    }
}
```

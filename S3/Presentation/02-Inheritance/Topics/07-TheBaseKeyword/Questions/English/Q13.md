## How does base work with Events?

### âœ… Answer:

`base` doesn't work **directly** with Events. We must use Virtual Methods.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public event EventHandler? Clicked;
    
    protected virtual void OnClicked() {
        Clicked?.Invoke(this, EventArgs.Empty);
    }
}

class Derived : Base {
    protected override void OnClicked() {
        base.OnClicked();  // âœ… Base event is raised
        // Custom logic
    }
}
```

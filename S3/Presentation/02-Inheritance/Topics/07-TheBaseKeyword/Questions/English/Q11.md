## How does base work with Indexers?

### âœ… Answer:

`base` also works with Indexers. We can call Base Indexer.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    protected int[] items = new int[10];
    
    public virtual int this[int index] {
        get => items[index];
        set => items[index] = value;
    }
}

class Derived : Base {
    public override int this[int index] {
        get => base[index] * 2;  // âœ… Base indexer
        set => base[index] = value / 2;
    }
}
```

## What is base keyword?

### âœ… Answer:

`base` keyword allows us to access **base class methods, properties and members**.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public void Method() {
        Console.WriteLine("Base");
    }
}

class Derived : Base {
    public void Method() {
        base.Method();  // âœ… Call Base method
        Console.WriteLine("Derived");
    }
}
```

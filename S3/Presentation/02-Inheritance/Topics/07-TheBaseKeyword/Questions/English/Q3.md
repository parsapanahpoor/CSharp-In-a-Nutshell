## How does base work in Override Methods?

### âœ… Answer:

`base` in Override Methods is used to **call Base Class method**.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public virtual void Method() {
        Console.WriteLine("Base");
    }
}

class Derived : Base {
    public override void Method() {
        base.Method();  // âœ… Base method
        Console.WriteLine("Derived");
    }
}
```

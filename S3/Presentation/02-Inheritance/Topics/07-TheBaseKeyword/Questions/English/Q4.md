## Why don't we have base.base?

### âœ… Answer:

Because `base` only refers to **direct Base Class**. We cannot access GrandParent.

<div dir="ltr" style="text-align: left;">

```csharp
class GrandParent {
    public virtual void Method() { }
}

class Parent : GrandParent {
    public override void Method() { }
}

class Child : Parent {
    public override void Method() {
        base.Method();  // âœ… Parent.Method()
        // base.base.Method();  // âŒ Error! base.base doesn't exist
    }
}
```

<div dir="ltr" style="text-align: left;">

### âœ… Solution:

<div dir="ltr" style="text-align: left;">

```csharp

```

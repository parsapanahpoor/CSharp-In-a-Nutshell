## How does base work in Constructor?

### âœ… Answer:

`base` in Constructor is used to **call Base Class Constructor**.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public Base(string name) {
        Console.WriteLine($"Base: {name}");
    }
}

class Derived : Base {
    public Derived(string name) : base(name) {  // âœ… Base constructor
        Console.WriteLine("Derived");
    }
}
```

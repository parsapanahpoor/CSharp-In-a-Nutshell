## Why doesn't Hiding have Polymorphism?

### âœ… Answer:

Because Hiding is resolved at **Compile-time**, not Runtime. Compiler decides based on **reference type**.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public void Method() {
        Console.WriteLine("Base");
    }
}

class Derived : Base {
    public new void Method() {
        Console.WriteLine("Derived");
    }
}

Base obj = new Derived();
obj.Method();  // Compiler sees: obj is type Base
               // So calls Base.Method() âœ…
               // Not Derived.Method() âŒ
```

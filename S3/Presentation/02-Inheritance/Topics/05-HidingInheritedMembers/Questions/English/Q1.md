## What is Member Hiding?

### âœ… Answer:

**Member Hiding** means defining a member with the **same name** in derived class that **hides** the base class member.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public void Display() {
        Console.WriteLine("Base Display");
    }
}

class Derived : Base {
    public new void Display() {  // âœ… Hiding
        Console.WriteLine("Derived Display");
    }
}
```

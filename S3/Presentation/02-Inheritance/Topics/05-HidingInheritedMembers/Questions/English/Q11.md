## Can we access Base Member after Hiding?

### âœ… Answer:

Yes! With **base** keyword we can access Base Member.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public void Method() {
        Console.WriteLine("Base");
    }
}

class Derived : Base {
    public new void Method() {
        base.Method();  // âœ… Call Base method
        Console.WriteLine("Derived");
    }
}

Derived d = new Derived();
d.Method();
// Output:
// Base
// Derived
```

## ‚ùì Q19:

## How does Hiding work with Reflection and Runtime Type Information?

### ‚úÖ Deep Answer:

Reflection can inspect Hidden Members at runtime and access Base Members.

### üîç Reflection for Hidden Members:

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public void Method() {
        Console.WriteLine("Base");
    }
}

class Derived : Base {
    public new void Method() {
        Console.WriteLine("Derived");
    }
}

// Reflection:
Type baseType = typeof(Base);
Type derivedType = typeof(Derived);

// Check Methods:
MethodInfo[] baseMethods = baseType.GetMethods();
MethodInfo[] derivedMethods = derivedType.GetMethods();

// Find Hidden Method:
MethodInfo baseMethod = baseType.GetMethod("Method");
MethodInfo derivedMethod = derivedType.GetMethod("Method");

// Check Hiding:
bool isHiding = derivedMethod.GetBaseDefinition() != derivedMethod;
Console.WriteLine($"Is Hiding: {isHiding}");  // True ‚úÖ
```

<div dir="ltr" style="text-align: left;">

### üî¨ Invoking Hidden Methods:

<div dir="ltr" style="text-align: left;">

```csharp
Derived obj = new Derived();

// With Reflection we can call Base Method:
MethodInfo baseMethod = typeof(Base).GetMethod("Method");
baseMethod.Invoke(obj, null);  // "Base" ‚úÖ

// Or Derived Method:
MethodInfo derivedMethod = typeof(Derived).GetMethod("Method");
derivedMethod.Invoke(obj, null);  // "Derived" ‚úÖ
```

<div dir="ltr" style="text-align: left;">

### üîç GetBaseDefinition():

<div dir="ltr" style="text-align: left;">

```csharp
MethodInfo method = typeof(Derived).GetMethod("Method");
MethodInfo baseDefinition = method.GetBaseDefinition();

Console.WriteLine(baseDefinition.DeclaringType);  // Base ‚úÖ
// GetBaseDefinition() points to Base Method
```

### üéØ Advanced Note:

Reflection can find **Hidden Members** and access **Base Members**. This is useful for **Code Analysis** and **Testing**.

### ‚ö†Ô∏è Performance:

Reflection for Hidden Members is **very slow** (~1000-10000 CPU cycles). Only use for **Inspection**, not for **Invocation**!

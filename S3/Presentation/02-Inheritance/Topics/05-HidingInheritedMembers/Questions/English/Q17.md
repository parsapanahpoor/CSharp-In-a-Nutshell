## ‚ùì Q17:

## How does Hiding work with Generic Types and Type Parameters?

### ‚úÖ Deep Answer:

Hiding in Generic Classes can get very complex, especially when combined with Type Constraints.

### üîç Complex Example:

<div dir="ltr" style="text-align: left;">

```csharp
class Base<T> {
    public void Process(T item) {
        Console.WriteLine($"Base processing {typeof(T).Name}");
    }
}

class Derived<T> : Base<T> {
    public new void Process(T item) {  // ‚úÖ Hiding
        Console.WriteLine($"Derived processing {typeof(T).Name}");
    }
}

// Usage:
Base<int> b = new Derived<int>();
b.Process(42);  // "Base processing Int32" ‚úÖ

Derived<int> d = new Derived<int>();
d.Process(42);  // "Derived processing Int32" ‚úÖ
```

<div dir="ltr" style="text-align: left;">

### ‚ö†Ô∏è Problem with Covariance:

<div dir="ltr" style="text-align: left;">

```csharp
class Animal { }
class Dog : Animal { }

class Base<T> {
    public void Process(T item) { }
}

class Derived<T> : Base<T> {
    public new void Process(T item) { }
}

// ‚ùå Cannot do this:
// Base<Animal> b = new Derived<Dog>();  // Error: Generic types invariant

// But with Interface Covariance:
interface IProcessor<out T> {
    void Process(T item);
}

class Processor<T> : IProcessor<T> {
    public void Process(T item) { }
}

// Now Covariance works:
IProcessor<Animal> p = new Processor<Dog>();  // ‚úÖ
```

### üî¨ Hiding with Type Constraints:

<div dir="ltr" style="text-align: left;">

```csharp
class Animal { }
class Dog : Animal { }

class Base<T> where T : Animal {
    public void Process(T animal) {
        animal.Eat();  // Knows T : Animal
    }
}

class Derived<T> : Base<T> where T : Animal {
    public new void Process(T animal) {
        base.Process(animal);  // ‚úÖ Base method
        // Can call Animal methods
    }
}
```

<div dir="ltr" style="text-align: left;">

### üéØ Advanced Note:

Hiding in Generic Classes preserves **Type Safety**. Compiler checks that Type Constraints are correct.

### ‚ö†Ô∏è Limitation:

Hiding cannot change Type Constraints. Must have the same Constraints.

## Why is `new` keyword necessary?

### âœ… Answer:

`new` keyword tells the compiler that we are **intentionally** hiding the member. Without `new`, compiler gives **warning**.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public void Method() { }
}

class Derived : Base {
    // âš ï¸ Warning: 'Derived.Method()' hides inherited member 'Base.Method()'
    public void Method() { }  // Warning!
    
    // âœ… With new:
    public new void Method() { }  // No warning
}
```

## How does Hiding work in Multi-Level Inheritance?

### âœ… Answer:

Hiding is done at each level and only hides **one level up**.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    public void Method() {
        Console.WriteLine("Base");
    }
}

class Derived : Base {
    public new void Method() {
        Console.WriteLine("Derived");
    }
}

class Derived2 : Derived {
    public new void Method() {
        Console.WriteLine("Derived2");
    }
}

Base b = new Derived2();
b.Method();  // "Base" âœ…

Derived d = new Derived2();
d.Method();  // "Derived" âœ…

Derived2 d2 = new Derived2();
d2.Method();  // "Derived2" âœ…
```

## When should we use Hiding?

### âœ… Answer:

Only when **really necessary** and we cannot use virtual/override.

### Example: When Base Class is from a Library

<div dir="ltr" style="text-align: left;">

```csharp

class Base {  // From Library
    public void Method() { }  // Not virtual!
}

// We cannot change Base:
class Derived : Base {
    public new void Method() {  // âœ… Must hide
        // Custom implementation
    }
}
```

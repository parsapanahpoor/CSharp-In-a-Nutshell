## How does Hiding work with Indexers?

### âœ… Answer:

Indexers can also be hidden, like Methods and Properties.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    protected int[] items = new int[10];
    
    public virtual int this[int index] {
        get => items[index];
        set => items[index] = value;
    }
}

class Derived : Base {
    public new int this[int index] {  // âœ… Hiding indexer
        get => items[index] * 2;
        set => items[index] = value / 2;
    }
}

Base b = new Derived();
b[0] = 10;  // Base indexer âœ…

Derived d = new Derived();
d[0] = 10;  // Derived indexer (value = 5) âœ…
```

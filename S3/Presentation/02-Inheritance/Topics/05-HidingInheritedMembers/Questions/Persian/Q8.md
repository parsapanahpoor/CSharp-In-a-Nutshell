## ❓ Q8:

## Hiding با Access Modifier چطور کار می‌کنه؟

### ✅ جواب:

Access Modifier در Hiding می‌تونه تغییر کنه، اما باید **کمتر محدودکننده** باشه.

<div dir="ltr" style="text-align: left;">

```csharp
class Base {
    protected void Method() { }  // Protected
}

class Derived : Base {
    public new void Method() { }  // ✅ Public (کمتر محدودکننده)
}

// ❌ نمی‌تونیم این کار رو بکنیم:
// private new void Method() { }  // Error! بیشتر محدودکننده
```

<div dir="rtl" style="text-align: right;">

### 🎯 نکته:

Access Modifier در Hiding باید **کمتر محدودکننده** باشه یا **همان** باشه. نمی‌تونه بیشتر محدودکننده باشه.

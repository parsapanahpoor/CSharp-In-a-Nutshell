## Can Abstract Class inherit from Interface?

### âœ… Answer:

Yes! Abstract Class can **implement** Interface.

<div dir="ltr" style="text-align: left;">

```csharp
interface IFlyable {
    void Fly();
}

abstract class Animal : IFlyable {
    public abstract void MakeSound();
    
    // Can make Fly abstract:
    public abstract void Fly();  // âœ… From IFlyable
    
    // Or implement:
    // public virtual void Fly() {
    //     Console.WriteLine("Flying...");
    // }
}

class Bird : Animal {
    public override void MakeSound() {
        Console.WriteLine("Tweet!");
    }
    
    public override void Fly() {  // âœ… Must implement
        Console.WriteLine("Bird is flying");
    }
}
```

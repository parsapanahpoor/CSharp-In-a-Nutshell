## What is Abstract Property?

### âœ… Answer:

**Abstract Property** works like Abstract Method. Must be implemented in derived class.

<div dir="ltr" style="text-align: left;">

```csharp
abstract class Shape {
    public abstract double Area { get; }  // âœ… Abstract property
}

class Circle : Shape {
    private double radius;
    
    public override double Area {  // âœ… Must implement
        get => Math.PI * radius * radius;
    }
}
```

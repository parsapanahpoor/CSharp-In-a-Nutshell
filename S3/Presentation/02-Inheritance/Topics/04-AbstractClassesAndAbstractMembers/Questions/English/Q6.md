## Can Abstract Class have both Abstract and Concrete Methods?

### âœ… Answer:

Yes! Abstract Class can have **both abstract and concrete methods**.

<div dir="ltr" style="text-align: left;">

```csharp
abstract class Animal {
    // Abstract method (must be implemented):
    public abstract void MakeSound();
    
    // Concrete method (can be used):
    public void Sleep() {
        Console.WriteLine("Sleeping...");
    }
    
    public void Eat() {
        Console.WriteLine("Eating...");
    }
}

class Dog : Animal {
    public override void MakeSound() {  // âœ… Must implement
        Console.WriteLine("Woof!");
    }
    // Inherits Sleep() and Eat() âœ…
}
```

## Can we convert Abstract Class to Concrete?

### âœ… Answer:

Yes! We can implement all Abstract Methods and make the class Concrete.

<div dir="ltr" style="text-align: left;">

```csharp
abstract class Animal {
    public abstract void MakeSound();
}

// Can create a Concrete Implementation:
class DefaultAnimal : Animal {
    public override void MakeSound() {
        Console.WriteLine("Default animal sound");
    }
}

// Now DefaultAnimal is a Concrete Class âœ…
```

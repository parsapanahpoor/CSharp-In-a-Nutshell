## How do Abstract Methods work with Events?

### âœ… Answer:

Events cannot be abstract, but we can use **Abstract Methods** to raise events.

<div dir="ltr" style="text-align: left;">

```csharp
abstract class Animal {
    public event EventHandler? Spoke;
    
    protected abstract void OnSpoke();  // âœ… Abstract method
    
    public void Speak() {
        Console.WriteLine("Speaking...");
        OnSpoke();  // âœ… Abstract method is called
    }
}

class Dog : Animal {
    protected override void OnSpoke() {  // âœ… Must implement
        Spoke?.Invoke(this, EventArgs.Empty);
    }
}
```

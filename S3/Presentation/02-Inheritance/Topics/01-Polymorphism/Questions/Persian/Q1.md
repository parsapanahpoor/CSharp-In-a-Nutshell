## ❓ Q1:

## Polymorphism یعنی چی؟

### ✅ جواب:

**Polymorphism** یعنی "چندریختی" یا "اشکال متعدد" - توانایی یک شی برای گرفتن فرم‌های مختلف.

### مثال ساده:

<div dir="ltr" style="text-align: left;">

```csharp
// یک رفرنس Animal می‌تونه اشیای مختلف نگه داره:
Animal animal1 = new Dog();    // ✅
Animal animal2 = new Cat();    // ✅
Animal animal3 = new Bird();   // ✅

// همه این‌ها Animal هستند، اما رفتار متفاوتی دارند!
```

<div dir="rtl" style="text-align: right;">

### 🎯 نکته:

Polymorphism به ما اجازه می‌ده با یک رابط واحد (مثل `Animal`) با انواع مختلف (Dog, Cat, Bird) کار کنیم.

## ❓ Q2:

## چند نوع Polymorphism داریم؟

### ✅ جواب:

دو نوع اصلی:

### 1️⃣ Static Polymorphism (چندریختی ایستا)
- در **زمان کامپایل** تصمیم گرفته می‌شود
- مثال: Method Overloading

### 2️⃣ Dynamic Polymorphism (چندریختی پویا)
- در **زمان اجرا** تصمیم گرفته می‌شود
- مثال: Method Overriding

<div dir="ltr" style="text-align: left;">

```csharp
// Static (Compile-time)
class Calculator {
    public int Add(int a, int b) => a + b;
    public double Add(double a, double b) => a + b;  // Overload
}

// Dynamic (Runtime)
class Animal {
    public virtual void Speak() { }
}
class Dog : Animal {
    public override void Speak() => Console.WriteLine("Woof!");
}
```

<div dir="rtl" style="text-align: right;">

### 🎯 نکته:

Static = کامپایلر تصمیم می‌گیره | Dynamic = زمان اجرا تصمیم می‌گیره

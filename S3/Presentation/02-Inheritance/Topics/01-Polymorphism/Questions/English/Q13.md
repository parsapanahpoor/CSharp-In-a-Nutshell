## Why is Polymorphism important for Design Patterns?

### âœ… Answer:

Most Design Patterns are based on Polymorphism!

### Example: Strategy Pattern

<div dir="ltr" style="text-align: left;">

```csharp
// Strategy Interface
interface IPaymentMethod {
    void Pay(double amount);
}

// Concrete Strategies
class CreditCard : IPaymentMethod {
    public void Pay(double amount) => 
        Console.WriteLine($"Paid ${amount} with Credit Card");
}

class PayPal : IPaymentMethod {
    public void Pay(double amount) => 
        Console.WriteLine($"Paid ${amount} with PayPal");
}

// Context
class ShoppingCart {
    private IPaymentMethod paymentMethod;
    
    public void SetPaymentMethod(IPaymentMethod method) {
        paymentMethod = method;  // âœ… Polymorphism
    }
    
    public void Checkout(double total) {
        paymentMethod.Pay(total);  // âœ… Each payment method knows how to pay
    }
}

// Usage:
var cart = new ShoppingCart();
cart.SetPaymentMethod(new CreditCard());
cart.Checkout(100);  // âœ…

cart.SetPaymentMethod(new PayPal());
cart.Checkout(200);  // âœ…
```

## What is Method Overriding?

### ‚úÖ Answer:

Method Overriding means **rewriting** the base class method in the derived class.

<div dir="ltr" style="text-align: left;">

```csharp
class Animal {
    public virtual void Speak() {
        Console.WriteLine("Animal makes sound");
    }
}

class Dog : Animal {
    public override void Speak() {  // ‚úÖ Override
        Console.WriteLine("Woof! Woof!");
    }
}

class Cat : Animal {
    public override void Speak() {  // ‚úÖ Override
        Console.WriteLine("Meow!");
    }
}

Animal animal1 = new Dog();
animal1.Speak();  // Output: "Woof! Woof!"

Animal animal2 = new Cat();
animal2.Speak();  // Output: "Meow!"
```

<div dir="ltr" style="text-align: left;">

### üéØ Note:

- Base method must be `virtual`
- Derived method must have `override`
- Decision is made at **runtime**

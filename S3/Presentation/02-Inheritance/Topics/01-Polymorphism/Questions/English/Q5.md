## What is the difference between Overloading and Overriding?

### âœ… Comparison:

| Feature | Overloading | Overriding |
|-------|-------------|------------|
| **Decision Time** | Compile-time | Runtime |
| **Class** | One class | Base + Derived |
| **Method Name** | Same | Same |
| **Parameters** | Must be different | Must be same |
| **Keyword** | Not needed | `virtual` + `override` |

<div dir="ltr" style="text-align: left;">

```csharp
// Overloading (one class)
class Math {
    public int Add(int a, int b) => a + b;
    public double Add(double a, double b) => a + b;  // âœ…
}

// Overriding (two classes)
class Animal {
    public virtual void Speak() { }
}
class Dog : Animal {
    public override void Speak() { }  // âœ…
}
```

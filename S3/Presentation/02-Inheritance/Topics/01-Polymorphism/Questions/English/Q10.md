## How does Polymorphism work in practice?

### ‚úÖ Practical Example:

<div dir="ltr" style="text-align: left;">

```csharp
class Shape {
    public virtual double GetArea() => 0;
}

class Circle : Shape {
    private double radius = 5;
    public override double GetArea() => Math.PI * radius * radius;
}

class Rectangle : Shape {
    private double width = 10, height = 5;
    public override double GetArea() => width * height;
}

// An array of Shapes:
Shape[] shapes = {
    new Circle(),
    new Rectangle(),
    new Circle()
};

// Process all without knowing exactly what type they are:
foreach (Shape shape in shapes) {
    Console.WriteLine($"Area: {shape.GetArea()}");  
    // Each one knows how to calculate its area!
}
```

<div dir="ltr" style="text-align: left;">

### üéØ Note:

This is the power of Polymorphism: **one code, multiple behaviors** based on the actual type of the object.
